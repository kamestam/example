Attributes
  • The h1 - h6 element defines headings.
  • The p element defines a paragraph.
  • The pre element defines preformatted text.
  • The mark element defines marked or highlighted text.
  • The strong element defines strong, semantic importance text.
  • The em element defines emphasized, semantic importance text.
  • Only the content inside the body section is displayed in a browser.
  • You can use either .htm or .html as file extension. No difference.
  • HTML elements with no content are called empty elements.
  • Attributes provide additional information about an element.
  • Browsers automatically add some white margin for heading.
